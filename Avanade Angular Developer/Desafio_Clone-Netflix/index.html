Zona de Combate (2021)

Em Zona de Combate, o piloto de drone Harp (Damson Idris) é enviado para uma perigosa zona militar. Lá, ele deve rastrear um dispositivo com grande poder destrutivo, sob a supervisão do oficial Leo (Anthony Mackie), antes que o aparato caia nas mãos erradas.
